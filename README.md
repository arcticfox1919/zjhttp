# zjhttp
最简HTTP服务器，C语言实现的简单http服务器，主要用于学习http协议和C语言的编程

Can run on Linux platform or Windows platform.


**使用**

在windows平台上，则将makefile文件名后面`_win32`去掉，然后cd到源码目录，输入make命令即可编译

Linux平台类似，去除`_linux`，使用make命令编译
